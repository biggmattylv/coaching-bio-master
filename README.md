# CoachingBio.com - Deployment Instructions

## How to "Drop It In"
1. **Locate this folder:** `C:\Users\Sophie\.openclaw\workspace\CoachingBio.com\dist`
2. **Deploy to Netlify:**
   - Log into [Netlify](https://app.netlify.com/).
   - Go to "Sites".
   - Drag and drop this `dist` folder directly onto the "Deploy" panel.
   - Netlify will instantly give you a live URL.

## What's Inside:
- `index.html`: Your master layout.
- `style.css`: The "Dodger Blue" and "Pop Magenta" theme.
- `assets/`: Folder for your images and icons.
