# "Playbook" Intake Logic

1. **Client Signs Up:** Fills out simple "Rookie" Intake form (Name, Email, Sport).
2. **"Bio Card" Generated:** System auto-creates the free page.
3. **The "All-Star" Prompt:** 24 hours later, system sends email: "Ready to take payments or fundraise?"
4. **Upgrade Logic:** Client clicks "All-Star," system triggers the payment/calendar integration workflow.
5. **Presence Management:** If they choose "Hall of Fame," system tags them for "Socialmonials" campaign.
